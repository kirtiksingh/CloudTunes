# CloudTunes

This Flutter based project allows for users to listen to music files stored in their Google Drive without downloading them on their mobile phones by providing or a Music Player app.

## Description

This Flutter based project allows for users to listen to music files stored in their Google Drive without downloading them on their mobile phones by providing or a Music Player app.

## Getting Started
